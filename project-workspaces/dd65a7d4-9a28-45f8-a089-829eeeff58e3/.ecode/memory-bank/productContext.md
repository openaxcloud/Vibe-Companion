# Product Context

## Problem Statement
Developers and technical writers need a blog platform that excels at displaying code snippets with proper syntax highlighting while maintaining simplicity and performance. Existing solutions often lack developer-focused features or require complex CMS setups.

## Target Users
- Software developers sharing technical knowledge
- Technical writers documenting projects
- Engineering teams publishing updates
- Open source maintainers writing tutorials

## UX Goals
- Distraction-free reading experience with clean typography
- Fast navigation between posts and categories
- Easy content discovery through search and tags
- Mobile-optimized reading experience
- Quick loading times for code-heavy posts

## Key User Flows
1. **Reader Flow**: Landing page → Browse posts → Read article with highlighted code → Subscribe to RSS
2. **Author Flow**: Write markdown → Preview with syntax highlighting → Publish → Verify RSS update
3. **Discovery Flow**: Search posts → Filter by tags → Navigate related content
