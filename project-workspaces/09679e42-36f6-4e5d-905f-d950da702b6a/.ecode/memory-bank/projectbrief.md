# Memory Bank - Intelligent AI Chatbot

## Project Overview
Memory Bank is an advanced AI chatbot platform featuring GPT-4.1 integration with persistent conversation memory, document-based knowledge retrieval, and code execution capabilities.

## Core Requirements
- OpenAI GPT-4.1 integration with streaming responses
- Conversation memory and context persistence
- Document upload for RAG (Retrieval-Augmented Generation) knowledge base
- Secure code execution sandbox environment
- Real-time typing animations for streaming responses
- Searchable conversation history with full-text search
- PDF export functionality for conversations
- Multi-language support (i18n)

## Goals
- Create an intelligent assistant that learns from uploaded documents
- Provide seamless conversation experience with memory retention
- Enable safe code execution and testing within conversations
- Support global users with localization

## Scope
Full-stack TypeScript React application with backend API, vector database for RAG, and containerized code execution environment.
