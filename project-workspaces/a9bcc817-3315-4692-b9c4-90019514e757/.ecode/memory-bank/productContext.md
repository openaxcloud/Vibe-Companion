# Product Context - Memory Bank Messaging

## Problem Statement
Teams need efficient real-time communication tools that support both casual conversation and structured collaboration, with reliable message delivery and rich interaction features.

## Target Users
- **Primary**: Development teams, remote workers, project collaborators
- **Secondary**: Small to medium businesses, community groups
- **User personas**: Team leads managing projects, developers sharing code snippets, remote workers staying connected

## UX Goals
- **Instant communication** with zero perceived latency
- **Organized conversations** through channels and threading
- **Rich context** via file sharing and message reactions
- **Awareness** of team member availability and message status

## Key User Flows
1. **Join workspace** → Browse channels → Start messaging
2. **Create channel** → Invite members → Share files/images
3. **Direct message** → Thread conversation → React to messages
4. **File upload** → Preview → Share with typing indicators
5. **Mobile notifications** → Quick reply → Mark as read
