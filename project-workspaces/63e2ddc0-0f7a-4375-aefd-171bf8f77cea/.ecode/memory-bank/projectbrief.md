# Restaurant Website Project

## Overview
Full-stack restaurant website featuring online menu browsing, table reservation system, and chef's blog for culinary insights and updates.

## Core Requirements
- Interactive menu with categories, pricing, and dietary filters
- Real-time table reservation system with availability checking
- Chef's blog with rich content management
- Mobile-responsive design for dine-in customers
- Admin dashboard for menu/reservation management

## Goals
- Increase online reservations by 40%
- Reduce phone booking load on staff
- Showcase restaurant personality through chef content
- Improve customer engagement pre-visit

## Scope
- Public website with menu, reservations, blog
- Admin panel for content management
- Email notifications for reservations
- Basic analytics dashboard
