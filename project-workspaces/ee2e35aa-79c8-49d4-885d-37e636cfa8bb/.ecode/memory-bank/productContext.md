# Product Context

## Problem Statement
Small businesses need an affordable, feature-rich e-commerce platform that handles payments, inventory, and customer management without technical complexity.

## Target Users
- **Primary**: Small business owners selling physical products
- **Secondary**: Customers seeking curated marketplace experience
- **Admin**: Store managers handling orders and inventory

## UX Goals
- Intuitive product discovery with smart search/filters
- Frictionless checkout with guest and registered user flows
- Clear order tracking and communication
- Seamless mobile shopping experience

## Key User Flows
1. Product browsing → Add to cart → Checkout → Payment → Confirmation
2. User registration/login → Profile management → Order history
3. Vendor dashboard → Product management → Order fulfillment → Analytics
4. Admin panel → Inventory tracking → Customer support → Platform management
