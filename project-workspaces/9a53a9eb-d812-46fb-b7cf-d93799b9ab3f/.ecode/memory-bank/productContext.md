# Product Context

## Problem Statement
Developers and technical writers need a simple, fast blog platform that excels at displaying code snippets and technical content without the overhead of complex CMS systems.

## Target Users
- Software developers sharing tutorials and insights
- Technical writers documenting projects
- Personal brand builders in tech

## UX Goals
- Instant page loads with static generation
- Excellent code readability with syntax highlighting
- Clean, distraction-free reading experience
- Easy navigation between posts
- Mobile-first responsive design

## Key User Flows
1. **Reader Discovery**: Landing page → Browse posts → Read article → Subscribe to RSS
2. **Content Consumption**: Post page → Read content → Navigate to related posts
3. **Code Reference**: Search for technical topic → Find relevant post → Copy code snippets
4. **RSS Subscription**: Discover blog → Subscribe to RSS → Receive updates
