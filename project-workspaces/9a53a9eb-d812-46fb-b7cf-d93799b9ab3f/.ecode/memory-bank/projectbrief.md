# Personal Blog Project

## Overview
A modern personal blog platform built with TypeScript and React, featuring markdown content authoring, syntax highlighting for code blocks, and RSS feed generation.

## Core Requirements
- Markdown file parsing and rendering with frontmatter support
- Syntax highlighting for code blocks using Prism.js or similar
- RSS feed generation for blog posts
- Responsive design for desktop and mobile
- Static site generation for optimal performance
- SEO optimization with meta tags and structured data

## Goals
- Create a fast, lightweight blogging platform
- Enable easy content creation through markdown
- Provide excellent developer experience for code sharing
- Ensure discoverability through RSS feeds

## Scope
- Blog post listing and individual post pages
- About page and contact information
- RSS feed endpoint
- Basic search functionality
- Dark/light theme toggle
