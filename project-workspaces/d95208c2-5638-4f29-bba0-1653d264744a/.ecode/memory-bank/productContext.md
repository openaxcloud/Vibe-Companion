# Product Context

## Problem Statement
Need a personal blog platform that supports technical writing with markdown, provides excellent code syntax highlighting, and offers RSS feeds for content distribution without the complexity of a full CMS.

## Target Users
- Technical writers and developers
- Content creators who prefer markdown
- Readers who consume technical content
- RSS feed subscribers

## UX Goals
- Fast page loads and smooth navigation
- Excellent reading experience on all devices
- Clean, distraction-free design
- Easy content discovery through categories/tags

## Key User Flows
1. **Reader Flow**: Homepage → Post list → Individual post → Related posts
2. **RSS Flow**: RSS client → Feed URL → New post notifications
3. **Content Discovery**: Search/filter posts by tags or categories
4. **Mobile Reading**: Optimized typography and layout for mobile devices
