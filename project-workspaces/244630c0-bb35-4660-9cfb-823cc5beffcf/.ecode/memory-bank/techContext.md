# Tech Stack

## Frontend
- **React 18** with TypeScript
- **Vite** for build tooling
- **Tailwind CSS** for styling with dark mode
- **React Query** for server state management
- **React Hook Form** with Zod validation
- **Stripe Elements** for payment forms

## Backend
- **Node.js/Express** with TypeScript
- **Prisma ORM** with PostgreSQL
- **Redis** for session storage and caching
- **Stripe API** for payments and webhooks
- **Nodemailer/SendGrid** for email notifications
- **Multer/AWS SDK** for file uploads

DATABASE_URL, REDIS_URL, STRIPE_SECRET_KEY, STRIPE_WEBHOOK_SECRET
