# Project Brief – E‑commerce Marketplace (Memory Bank)

1. **Project Name:** Aurora Market – Full‑stack TypeScript E‑commerce Marketplace.
2. **Objective:** Deliver a production‑ready marketplace with secure payments, robust catalog, and a smooth checkout, optimized for web and mobile.
3. **Core Features:**
   - Product catalog with search, category filters, sorting, and pagination.
   - Shopping cart and multi‑step checkout with Stripe payment integration.
   - User authentication (signup, login, password reset) and profile management.
   - Order management dashboard for buyers and admins.
   - Inventory tracking with stock updates on purchase and admin edits.
   - Email notifications for order confirmations and status updates.
   - Fully responsive UI with light/dark mode toggle.
4. **Scope (MVP):**
   - Single‑currency payments, minimal tax/shipping logic, single region.
   - Basic admin features: manage products, view orders, update statuses.
   - Essential security: HTTPS, JWT sessions, Stripe best practices.
5. **Non‑Goals (for now):**
   - Multi‑vendor payouts, subscriptions, marketplace commissions.
   - Advanced analytics, recommendation engine, loyalty programs.
6. **Success Criteria:**
   - A user can discover products, pay with Stripe, receive email confirmation, and view their order history from any device.
   - Admin can maintain the catalog and track orders without direct DB access.
   - System handles typical small‑store traffic reliably and securely.
