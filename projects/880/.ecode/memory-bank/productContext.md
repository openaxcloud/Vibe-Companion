# Product Context

## Problem Statement
Build a full-stack e-commerce marketplace with Stripe payments, product catalog with search and filters, shopping cart with checkout flow, user authentication, order management dashboard

## Target Users
- Primary users who need this functionality
- Secondary users who may benefit

## User Experience Goals
- Intuitive interface
- Fast performance
- Accessible design

## How It Should Work
1. User accesses the application
2. Core functionality is immediately available
3. Data is persisted across sessions
