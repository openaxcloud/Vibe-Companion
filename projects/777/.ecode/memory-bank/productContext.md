## Problem & Users
Teams >10 people lose context in email & Discord. Target: remote software teams, agencies, game dev studios.

### Pain Points
- Message loss on flaky mobile
- No reliable read receipts
- Cluttered threads & lost files

### UX Goals
- Sub-100 ms message roundtrip
- Always-visible presence dots
- One-click file preview & search

### Key Flows
1. Onboarding → create workspace → invite via link → join default #general
2. Channel creation → privacy toggle → add members → real-time sync
3. DM initiation → typeahead user search → instant WebSocket room
4. File drag-drop → upload progress → markdown embed → push notification
5. Thread creation → reply in-line → mark parent read → unread badge counts
