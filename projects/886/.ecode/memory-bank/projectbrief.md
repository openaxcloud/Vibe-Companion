# Project Brief

## Overview
Build an intelligent AI chatbot with OpenAI GPT-5 integration, conversation memory, document upload for RAG knowledge base, streaming responses

## Core Requirements
- Implement the core functionality described above
- Ensure responsive design for all screen sizes
- Follow best practices for typescript development

## Project Goals
- Deliver a working MVP
- Clean, maintainable code
- Good user experience

## Scope
- Frontend: react application
- Backend: API endpoints as needed
- Database: PostgreSQL for persistence
