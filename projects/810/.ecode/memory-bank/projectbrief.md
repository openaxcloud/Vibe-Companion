# Project Brief: Memory Bank – Simple Todo App

## Overview
Memory Bank is a minimalist todo application built in TypeScript with React, designed to help users quickly capture, organize, and track personal tasks.

## Core Requirements
- Add, edit, and delete todo items
- Mark todos as complete / incomplete
- Persist todos in local storage (per-browser memory bank)
- Simple filtering (All / Active / Completed)
- Clear completed items in bulk
- Responsive UI for desktop and mobile

## Goals
- Fast capture and recall of small personal tasks
- Zero-backend MVP using browser-based persistence
- Clean, distraction-free interface
- Codebase structured for easy future expansion (tags, due dates, sync)

## Scope (MVP)
- Single-user, single-device experience
- No authentication, no server API
- Core todo CRUD + local storage sync
- Basic accessibility (keyboard, screen reader friendly)
- Deployed as a static SPA (e.g., Vercel/Netlify)

## Non-Goals (for now)
- Multi-user accounts or collaboration
- Cross-device sync or cloud backup
- Advanced task management (projects, reminders, calendars)
- Native mobile apps
