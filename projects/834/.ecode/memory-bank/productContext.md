# Product Context – Memory Bank

## Problem Statement
- Need a baseline, reusable e‑commerce marketplace implementation that supports core selling flows without heavy custom dev each time.
- Business wants a secure, modern UX for browsing, buying, and managing orders with Stripe payments.

## Target Users
- Shoppers: browse products, search/filter, add to cart, pay securely, track orders.
- Store admins: manage products, orders, inventory, and basic settings.
- (Future) Vendors: manage their own listings and see sales data.

## UX Goals
- Fast, intuitive product discovery with powerful but simple filters.
- Frictionless checkout with clear progress and transparent pricing.
- Trust and confidence around payments and order status.
- Responsive design that works well on mobile and desktop.

## Key User Flows
- Browse catalog → apply filters/sort → view product → add to cart → checkout → pay (Stripe) → view confirmation.
- New user: sign up → verify (optional) → browse → purchase → view order history.
- Returning user: login → cart restored → reorder or continue shopping.
- Admin: login → access dashboard → view orders → update status → manage product listings.
