# Project Brief: Memory Bank – Todo App

## Overview
Memory Bank is a simple, focused todo application built with React and TypeScript. It serves as the initial product for a broader "Memory Bank" system that will eventually manage notes, tasks, and personal knowledge.

## Core Requirements
- Create, read, update, and delete (CRUD) todo items
- Persist todos across sessions (local storage for now)
- Support basic task metadata: title, description, status (pending/done)
- Allow simple filtering (e.g., all / active / completed)
- Provide a clean, minimal UI optimized for desktop web

## Goals
- Deliver a small but production-quality React + TypeScript codebase
- Establish patterns reusable for future Memory Bank modules (notes, journals, etc.)
- Keep the app fast, accessible, and easy to extend

## Scope (Initial Version)
- Single user, browser-only experience (no auth, no backend)
- Local persistence using `localStorage`
- One main screen: task list with inline create/edit
- Light theming, responsive layout for mobile/desktop
- No collaborative features, reminders, or calendar integration yet
