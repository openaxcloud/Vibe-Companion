### Product Context: Simple Task Management

**Problem Statement:**
Many users require a quick, no-frills tool to jot down and track simple tasks without the overhead of complex project management systems or account creation. Existing solutions can be overly complicated for basic, temporary task lists.

**Target Users:**
Individuals seeking a personal, temporary task management tool. This includes students, developers prototyping, or anyone needing a quick browser-based checklist.

**UX Goals:**
*   **Intuitive:** Users should immediately understand how to add, complete, and delete tasks.
*   **Responsive:** The interface should adapt well to different screen sizes (though not a primary focus, basic responsiveness is desired).
*   **Efficient:** Minimal clicks or interactions required to perform core actions.

**Key User Flows:**
1.  **Add Todo:** User types task into input field -> Clicks 'Add' button -> Task appears in list.
2.  **Mark Complete:** User clicks on a checkbox/item -> Task visually changes (e.g., strikethrough) -> User can toggle completion status.
3.  **Delete Todo:** User clicks a 'Delete' button next to a task -> Task is removed from the list.
