### Project Brief: Simple Todo App

This project aims to develop a straightforward, client-side Todo application. The primary goal is to provide users with a basic interface to manage tasks, focusing on core functionalities without external dependencies or complex features.

**Core Requirements:**
*   Add new todo items to a list.
*   Mark existing todo items as complete or incomplete.
*   Delete todo items from the list.

**Goals:**
*   Demonstrate foundational web development skills using HTML, CSS, and JavaScript.
*   Create a clean, functional user interface.
*   Ensure basic interactivity and responsiveness.

**Scope:**
*   Client-side application only.
*   No persistent storage (data will reset on page refresh).
*   Minimal styling for usability.
