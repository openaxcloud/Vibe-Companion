# Project Brief: Calculator App – Memory Bank

## Overview
- Build a web-based calculator app using **TypeScript + React**.
- This Memory Bank entry tracks the project vision, constraints, and core requirements.

## Core Requirements
- Perform basic arithmetic: add, subtract, multiply, divide.
- Support decimal numbers and clear/reset.
- Show current input and last computed result.
- Provide a clean, responsive UI suitable for desktop and mobile.

## Goals
- Deliver a reliable, easy-to-use calculator with predictable behavior.
- Keep architecture simple but structured for future extension (e.g., history, memory, scientific mode).
- Ensure codebase is type-safe, testable, and easy for others to contribute.

## Scope (Initial Version)
- One main calculator view with numeric keypad and operation buttons.
- Keyboard input support for common keys (0–9, +, -, *, /, Enter, Backspace) if feasible.
- Basic error handling (e.g., divide-by-zero, invalid expressions) surfaced in the UI.

## Out of Scope (Initial Version)
- Scientific functions (sin, cos, log, etc.).
- Advanced memory operations (M+, M-, MR, MC) beyond a possible simple last-result recall.
- Multi-theme support or complex settings panels.

## Success Criteria
- All basic operations work correctly and are covered by unit tests for the calculation logic.
- UI is intuitive, with clear feedback on operations and errors.
- Code structure supports adding new features with minimal refactoring.
